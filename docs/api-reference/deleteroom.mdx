## **DELETE** /**rooms/\{roomId\}**

```bash
curl --request DELETE \
  --url https://api.textchat.example.com/rooms/{roomId}
```

### **Path Parameters**

- **roomId**
    - **string**
    - **required**

**204**

Roomが削除されました
