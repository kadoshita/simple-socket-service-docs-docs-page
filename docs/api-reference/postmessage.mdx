## WEBHOOK /messages

### **Body**

**application/json**

- **message**
    - **string**
    - The message to be sent
- **roomId**
    - **string**
    - The room ID
- **clientId**
    - **string**
    - The client ID

```json
{
  "example": {
    "value": {
      "message": "<string>",
      "roomId": "<string>",
      "clientId": "<string>"
    }
  }
}
```

### **Response**

**200**

Return a 200 status to indicate that the data was received successfully
